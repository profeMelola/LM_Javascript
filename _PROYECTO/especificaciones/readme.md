# Especificaciones para la programación JS

## Sobre productos

Cada producto deberá tener los siguientes atributos:

- id
- nombre
- precio
- imagen
- info

Por ejemplo:

    {
        id: 1,
        nombre: 'Patata',
        precio: 1,
        imagen: 'imgs/patata.jpg',
        info: 'La patata es un alimento rico en hidratos de carbono'
    }
